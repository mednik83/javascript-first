# Habit Tracker Mini App

Мини-приложение для трекинга привычек.  

**Функции:**
- Добавление новых привычек с иконкой и целью
- Добавление комментариев к каждому дню привычки
- Удаление дней привычки
- Просмотр прогресса по привычкам

**Как запустить:**
1. Откройте проект в VS Code или другом редакторе.
2. Запустите `Live Server` на `index.html`.
3. Приложение откроется в браузере и готово к использованию.

> ⚠️ Работает только через Live Server или любой HTTP-сервер. Прямое открытие `index.html` через `file://` не поддерживается.
